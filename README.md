本地生活小助手（lin.py项目）代码分析与操作指南
一、项目基础准备步骤
步骤 1：创建项目文件
在本地任意目录下新建项目文件夹，在该项目文件夹内新建Python代码文件，命名为ruanjian.py。
步骤 2：复制完整代码到文件
1. 打开新建的ruanjian.py文件
2. 将上述完整的生活小助手代码复制粘贴到文件中，保存文件
步骤 3：运行项目并验证初始化
1. 打开终端/命令行工具（Windows用CMD/PowerShell，Mac/Linux用终端）
2. 通过cd命令切换到ruanjian.py所在的文件夹目录
3. 执行运行命令：python ruanjian.py
4. 首次运行效果：
   - 自动在项目文件夹内生成2个数据文件：
   1. life_helper_data.json：程序运行依赖的核心数据文件，用于持久化存储待办事项和记账数据
   2. life_helper_data.txt：纯文本可读格式文件，方便人工查看数据内容
   - 终端先显示TXT数据文件的绝对路径（提示：📄 可读数据文件将保存到：XXX）
   - 随后显示「本地生活小助手」主菜单界面，包含9大功能和退出选项，初始化完成
二、核心功能分步测试验证
功能1：待办事项管理（含自定义时间+新增+查看+标记完成+删除）
1. 添加待办（支持自定义时间）
   - 主菜单输入1→ 进入待办管理界面，输入1
   - 输入待办内容（如“明天上午9点开会”）
   - 输入y选择自定义时间，或输入n使用默认当前日期：
     - 选y：需输入格式为YYYY-MM-DD的时间（如2026-02-15），格式错误会循环提示重新输入，输入正确后确认时间
     - 选n：自动使用当前日期（如2026-01-02）
   - 提示“✅ 待办添加成功！”，同时自动保存JSON和TXT数据文件，终端显示TXT文件绝对路径
   - 打开life_helper_data.json或life_helper_data.txt，可看到待办事项新增对应记录
2. 查看待办
   - 待办管理界面输入2
   - 显示待办列表，包含序号、内容、创建/指定时间、状态（初始为“🔲 未完成”）
   - 若暂无待办，提示“暂无待办事项~”
3. 标记待办为完成
   - 待办管理界面输入3 → 先展示待办列表
   - 输入待办序号（如1，程序自动转换为索引，无需输入0）
   - 若该待办未完成，提示“✅ 待办标记为完成！”；若已完成，提示“⚠️  该待办已标记为完成，无需重复操作！”
   - 再次查看待办，对应状态变为“✅ 已完成”，数据自动同步保存到两个数据文件
4. 删除待办事项
   - 待办管理界面输入4 → 先展示待办列表
   - 输入要删除的待办序号（如1）
   - 程序弹出确认提示，输入y确认删除，提示“✅ 已成功删除待办「XXX」！”；输入n则取消删除
   - 若输入无效序号（如超出列表长度）或非数字，提示对应错误信息
   - 删除后数据自动同步保存，TXT文件同步更新
5. 异常场景测试：输入非1-4的选项（如5），提示“❌ 无效操作选项，请选择1-4！”
功能2：简易记账（添加支出+查看本月支出）
1. 添加支出
   - 主菜单输入2 → 进入记账界面，输入1
   - 输入支出项目（如“早餐-包子豆浆”）
   - 输入金额（如5.5，支持整数和小数）
   - 提示“✅ 记账成功！”，同时保存JSON和TXT数据文件，终端显示TXT文件绝对路径
   - 打开两个数据文件，可看到expenses数组/记账数据中新增对应记录
2. 查看本月支出
   - 记账界面输入2
   - 自动识别当前月份（如“2026-01”），显示该月所有支出明细（日期、项目、金额，金额保留2位小数）
   - 底部显示本月总支出金额，若本月无支出，总支出显示0.00元
功能3：随机日程推荐
   - 主菜单输入3
   - 程序从预设的5条日程模板中随机抽取1条展示（如“💡 今日推荐：花30分钟做一组居家运动（比如帕梅拉15分钟燃脂操）”）
   - 每次运行该功能，推荐内容可能不同（随机效果），无需保存数据
功能4：生成搜索链接
   - 主菜单输入4
   - 输入要搜索的内容（如“Python基础教程”，支持中文）
   - 程序自动生成百度搜索链接（已通过quote函数处理中文编码，避免链接失效）
   - 提示“复制链接到浏览器打开即可搜索”，粘贴链接到浏览器可直接跳转至对应搜索结果页
功能5：安全密码生成器
1. 主菜单输入5
2. 输入密码长度（建议8位及以上，如12；若输入小于4位，提示“❌ 密码长度建议不小于4位！”；若输入非数字，提示对应错误）
3. 输入y选择包含特殊字符，输入n则不包含
4. 程序自动生成密码：
   -  确保至少包含小写字母、大写字母、数字（若选择包含特殊字符，还会至少包含1个特殊字符）
   - 生成后打乱字符顺序，提高密码安全性，提示“✅ 生成的密码：XXX”
5. 注意：程序不存储生成的密码，需自行妥善记录
功能6：文本内容统计
1. 主菜单输入6
2. 输入文本文件路径（支持.txt和.md格式，如./test.txt，需确保文件存在于lin.py项目文件夹内；若文件在其他目录，输入绝对路径，如C:\Users\XXX\Desktop\test.txt）
3. 若文件不存在或格式不支持，提示“❌ 无效的文本文件（仅支持.txt/.md格式）！”
4. 程序读取文件内容，统计并输出4项数据：
   - 总字符数（含空格、换行、制表符）
   - 纯字符数（不含空格、换行、制表符）
   - 文本总行数
   - 单词数（按空格分割，过滤空字符串）
5. 异常场景：文件读取失败（如权限不足），提示“❌ 文件读取失败：XXX”
功能7：日期计算器（天数差计算/日期推算）
1. 计算两个日期的天数差
   - 主菜单输入7 → 输入1
   - 依次输入两个日期（格式必须为YYYY-MM-DD，如2026-01-01、2026-01-10；格式错误则提示对应信息）
   - 程序输出两个日期的绝对天数差（如“✅ 两个日期相差：9 天”）
2. 推算N天前/后的日期
   - 日期计算器界面输入2
   - 输入天数（正数=未来日期，负数=过去日期，如7或-3；非数字则提示错误）
   - 程序自动显示当前日期和目标日期（如“✅ 7 天后/前的日期：2026-01-09”）
功能8：凯撒密码加密/解密（含中文占位混淆）
1. 加密文本
   - 主菜单输入8 → 输入1
   - 输入要加密的文本（支持英文+中文+标点，如“Hello World! 你好”）
   - 输入偏移量（建议1-25，如3；非数字则提示错误）
   - 程序先对英文字母进行凯撒加密（其他字符不变），再随机插入中文占位符混淆，输出加密后文本
2. 解密文本
   - 凯撒密码界面输入2
   - 输入加密后的文本（无需剔除中文占位符，不影响英文还原）
   - 输入与加密时一致的偏移量（如3）
   - 程序自动还原英文字母，输出解密后文本（中文占位符保留，不影响核心内容读取）
3. 注意：该加密为简易加密，不适合高度机密信息传输
功能9：系统信息查询
   - 主菜单输入9
   - 程序自动查询并输出3项核心系统信息：
   - 操作系统类型（Windows/Linux/Mac OS/未知系统）
   - 当前登录用户名（无法获取时显示“无法获取”）
   - 当前工作目录（即ruanjian.py所在项目文件夹路径）
功能0：退出程序
   - 主菜单输入0
   - 程序提示“👋 程序已退出，再见！”，并正常终止运行
   - 所有待办事项和记账数据已自动持久化保存到life_helper_data.json和life_helper_data.txt，下次运行可读取历史数据
