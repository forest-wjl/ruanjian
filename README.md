步骤 1：创建项目文件
    在本地任意目录下新建文件夹，代码文件：命名为ruanjian.py
步骤 2：复制完整代码到文件
    1.打开ruanjian.py文件（用任意文本编辑器）
    2.复制GitHub中上传的完整代码，粘贴到文件中并保存
步骤 3：运行项目并验证功能
    1.打开终端 / 命令行工具，通过cd命令切换到该文件夹目录
    2.执行运行命令：python ruanjian.py
    3.首次运行将自动生成todo_list.txt数据文件（用于存储待办事项），并显示工具欢迎界面与功能菜单
    4.按以下测试用例验证核心功能（确保所有功能可正常使用）：
    (1)添加事项:
    输入 a → 输入事项名称（如 “完成作业”）→ 输入截止时间（如 “2026-1-1”,如果输入时间比今天早，那么会出现“警告：截止时间已过期，是否继续添加？(y/n)：”）
    可以发现：提示 “✅ 待办事项添加成功！”，打开todo_list.txt可看到新增记录0
    (2)查看事项	
    输入 v → 选择筛选 “全部”→ 选择排序 “添加顺序”	
    可以发现：显示统计信息（总计 1 项、已完成 0 项、未完成 1 项），并展示新增的待办事项	
    (3)标记完成	
    输入 m → 输入序号 1	
    可以发现：提示 “✅ 标记完成成功！”，再次查看事项时状态变为 “✅ 已完成”	
    (4)搜索事项	
    输入 s → 输入关键词 “作业”	
    可以发现：显示匹配结果（1 项，即 “完成作业”）
    (5)编辑事项	
    输入 e → 输入序号 1 → 选择 “1. 修改事项名称”→ 输入新名称 “完成人工智能作业 3”	
    可以发现：提示 “✅ 待办事项修改成功！”，查看事项时名称更新	
    (6)批量删除	
    输入 b → 选择 “2. 批量删除待办”→ 输入序号 1 → 确认 “y”	
    可以发现：提示 “✅ 成功删除 1 个事项！”，查看事项时显示 “当前暂无待办事项”		
    (7)退出工具	
    输入 q → 确认 “y”	
    可以发现：工具正常退出，所有数据自动保存到 todo_list.txt
